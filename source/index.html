Personal Projects

Zori

The best real-time music link sharer ever built. Uses Firebase as its backend and AngularJS on the frontend.

JS1K Dragonsnake

My entry into the JS1K 2014 contest. Dragonsnake is a cross between Fruit Ninja and Snake, made in less than 1K of JavaScript.

Smoov

Smoov is an intelligent public transportation app, which fully uses all smooth transportation means in Montpellier. Built with Sencha Touch and Node.js during the first Hackathon Opendata Montpellier, secund prize.

Panoramanode

A series of self-guided workshops to discover Node.js, in French. Built with Workshopper, inspired by Nodeschool.

D4

D4 is a visualization experiment. Built with d3.js and Bootstrap, it helps visualizing RDF DataCube datasets and their metadata.

SDMXDataCube

SDMX2DataCube is a Datalift module which converts SDMX data to RDF using the DataCube ontology. Built in JEE with KnockoutJS.
